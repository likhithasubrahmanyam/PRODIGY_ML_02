# PRODIGY_ML_02
# Customer Segmentation using K-Means Clustering

## Overview

This project implements the K-Means Clustering algorithm to segment retail store customers based on their purchasing behavior. By analyzing customer Annual Income and Spending Score, the model identifies distinct customer groups that can help businesses develop targeted marketing strategies and improve customer engagement.

## Project Objectives

- Perform customer segmentation using K-Means Clustering.
- Identify the optimal number of clusters using the Elbow Method.
- Visualize customer groups and cluster centroids.
- Gain business insights from customer purchasing patterns.

## Dataset

- **Dataset:** Mall Customers Dataset
- **Source:** https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python

## Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Workflow

1. Import required libraries
2. Load the dataset
3. Perform Exploratory Data Analysis (EDA)
4. Select relevant features
5. Determine the optimal number of clusters using the Elbow Method
6. Train the K-Means Clustering model
7. Visualize customer segments
8. Interpret customer groups

## Results

The K-Means algorithm successfully grouped customers into meaningful clusters based on their Annual Income and Spending Score. These customer segments can help businesses:

- Identify high-value customers
- Design personalized marketing campaigns
- Improve customer retention
- Optimize product recommendations
- Support business decision-making
